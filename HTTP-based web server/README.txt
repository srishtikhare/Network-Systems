------------------------------------------------------------------------------------------------------------------------------------------------------------------------
FILE FOLDER DESCRIPTION
For this assignment, I have created a single folder containing the following fies:
1) server code --> webserver.c
2) Makefile
3) Root folder and ws.conf file
------------------------------------------------------------------------------------------------------------------------------------------------------------------------

------------------------------------------------------------------------------------------------------------------------------------------------------------------------
USING THE MAKEFILE
You can complie the code using the Makefile using following commands:
1) make clear
2) make
------------------------------------------------------------------------------------------------------------------------------------------------------------------------

------------------------------------------------------------------------------------------------------------------------------------------------------------------------
GENERATING THE EXECUTABLE
This creates an executable file called "webserver".
------------------------------------------------------------------------------------------------------------------------------------------------------------------------

------------------------------------------------------------------------------------------------------------------------------------------------------------------------
HOW TO RUN THE CODE
It can be executed using command "./webserver".
------------------------------------------------------------------------------------------------------------------------------------------------------------------------

------------------------------------------------------------------------------------------------------------------------------------------------------------------------
ERRORS
1) Status Code "400" -- invalid request method
2) Status Code "400" -- invalid request url
3) Status Code "400" -- invalid request version
4) Status Code "404" -- requested URL does not exist (file does not exist)
5) Status Code "501" -- requested file not supported by the server
6) Status Code "500" -- cannot allocate memory
------------------------------------------------------------------------------------------------------------------------------------------------------------------------

------------------------------------------------------------------------------------------------------------------------------------------------------------------------
REQUEST METHODS IMPLEMENTED
I have implemented GET and POST request methods for this assignment.
------------------------------------------------------------------------------------------------------------------------------------------------------------------------

------------------------------------------------------------------------------------------------------------------------------------------------------------------------
PIPELINING
I have implemented the extra credit Pipelining feature in this assignment. This way, the web server supports persistent connections and pipelining of client requests.
"Connection: Keep-alive" headers have been used.
The timeout value is configurable in the server.
